OSC (Open Sound Control) Receiver Message Patterns

Transport:
  /play          ... start playback
  /pause         ... pause playback
  /record        ... start playback with recording enabled
  /seek   f time ... jump to time in seconds, range 0 - duration
  /rewind        ... jump to start (time 0)

ID control (index 1 to n):
  /mute     if ID mute      ... mute, 0 or 1
  /solo     if ID solo      ... solo, 0 or 1
  /pan/xy  iff ID x y       ... cartesian pos
  /pan/ae  iff ID azim elev ... polar pos in degrees
  /span     if ID span      ... span (virtual sound width), 0 to 1
  /gain/rms if ID gain      ... gain rms, 0 to 2, 1 unity
  /gain/db  if ID gain      ... gain dB, -100 to +12, 0 unity
  /set/xy ifff ID x y span  ... cartesian pos + span
  /set/ae ifff ID a e span  ... polar pos + span

ID recording (ID index 1 to n):
  /arm       if ID arm ... arm ID for recording, 0 or 1
  /start/xy iff ID x y ... start recording ID path at pos
  /pan/xy   iff ID x y ... move ID to pos
  /end/xy   iff ID x y ... stop recording ID path at pos
  /start/ae iff ID a e ... start recording ID path at polar pos
  /pan/ae   iff ID a e ... move ID to polar pos
  /end/ae   iff ID a e ... stop recording ID path at polar pos

Group recording (Group name string):
  /group/arm       sf name arm ... arm Group for recording, 0 or 1
  /group/start/xy sff name x y ... start recording Group path at pos
  /group/pan/xy   sff name x y ... move Group to pos
  /group/end/xy   sff name x y ... stop recording Group path at pos
  /group/start/ae sff name x y ... start recording Group at polar pos
  /group/pan/ae   sff name x y ... move Group to polar pos
  /group/end/ae   sff name x y ... stop recording Group at polar pos

Deprecated:
  /pan/az ifff ID-1 azim elev span ... Zirkonium MK II polar position,
                                       (ID index 0 to n-1, span 0 to 50)

Coordinate ranges:
  cartesian pos x: -1 to 1, 0 center & 1 right
  cartesian pos y: -1 to 1, 0 center & 1 front
  polar pos azim:  -180 to 180 or 0 to 360, 0 front & rotate clockwise
  polar pos elev:  0 to 180, 0 front & 90 above
